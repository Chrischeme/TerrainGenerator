To run, open the project with Visual Studio at Terrain Generation > Terrain Generation.sln. 
If ran, you should see the map created.

To change any user defined data, edit the textfile at Terrain Generation > Data > data.txt to hold the values
you want. 

To change the map width and map length, go to Terrain Generation > TextureHelper.h to change MAP_X and MAP_Z.

Code is at 
Terrain Generation > Source.cpp
Terrain Generation > SoftwareAgents.h
Terrain Generation > TextureHelper.h
